# SistemaBancario1

Sistema Bancario

El Sistema Bancario es una aplicación de consola que simula las operaciones de un banco real. Permite registrar clientes con validaciones estrictas (edad mínima, CURP único), gestionar múltiples tipos de cuentas (Ahorro, Cheques, Inversión) con saldos mínimos específicos, y realizar operaciones financieras como depósitos, retiros y transferencias. El sistema implementa reglas de negocio realistas (comisiones por retiro en cuentas de ahorro), mantiene un historial completo de movimientos y genera reportes estadísticos. Demuestra conceptos avanzados de POO incluyendo enumeraciones, propiedades calculadas, validaciones complejas y persistencia de datos.
